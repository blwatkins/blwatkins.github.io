# update the list of remote branches

This command will remove any remote branches that no longer exist in the remote repository.

```shell
git remote update --prune
```

```shell
git fetch --all
```

# update branches

```shell
git fetch
```

```shell
git fetch --all
```

```shell
git fetch --all --prune
```
